# EDA Analysis in Python
Complete Exploratory Data Analysis on the Titanic dataset. Includes data cleaning, visualization, and insights using Python, Pandas, Matplotlib, and Seaborn.

## 📊 Overview

This project analyzes the Titanic passenger dataset to understand survival patterns and key factors that influenced survival.

## 🛠️ Technologies Used

- Python 3
- Pandas (data manipulation)
- NumPy (numerical operations)
- Matplotlib (visualization)
- Seaborn (statistical visualization)

## 📈 Key Analyses

- Data cleaning and preprocessing
- Missing value handling
- Survival analysis by sex, class, age
- Correlation analysis
- Visualizations (bar charts, histograms, box plots, heatmaps)

## 🔍 Key Insights

| Factor | Finding |
|--------|---------|
| Sex | Women had significantly higher survival |
| Class | 1st class passengers survived more |
| Age | Children had better survival rates |
| Fare | Higher fare = higher survival |

## 🚀 How to Run

1. Open `Titanic_EDA_Project.ipynb` in Google Colab
2. Run all cells (Runtime → Run all)
3. View the analysis and visualizations
